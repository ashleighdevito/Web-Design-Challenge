Summary: Latitude vs X

The purpose of this project was to examine the relationship between weather and distance to the equator. Weather data used in this analysis was pulled from the OpenWeatherMap API from over 500 cities on 6/13/2021.


MatplotLib was used to create visualizations of the respective cities Lattitude as compared to Temperature, Humidity, Cloudiness, and Wind Speed.


The data provided evidence of conventional wisdom, that Temperature tended to increase as you approached the equator. The other weather factors did not have as visible of a relationship.


This site includes pages for each of the four plots, as well as a comparison page, and the original data used for analysis.
